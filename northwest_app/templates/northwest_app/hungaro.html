¿Qué es el método Húngaro?

El método Húngaro es un algoritmo de optimización combinatoria que resuelve problemas de asignación de manera óptima en tiempo polinomial. Fue desarrollado por el matemático húngaro Dénes Kőnig y posteriormente mejorado por Harold W. Kuhn.

Este método es especialmente eficiente para asignar n tareas a n agentes minimizando el costo total, garantizando que cada tarea sea asignada exactamente a un agente y cada agente reciba exactamente una tarea.

02

Pasos del algoritmo Húngaro

Paso 1: Reducción de filas

Restar el elemento mínimo de cada fila a todos los elementos de esa fila.

Paso 2: Reducción de columnas

Restar el elemento mínimo de cada columna a todos los elementos de esa columna.

Paso 3: Cobertura de ceros

Cubrir todos los ceros con el mínimo número de líneas (filas o columnas).

Paso 4: Prueba de optimalidad

Si el número de líneas es igual a n, se ha encontrado la asignación óptima.

Paso 5: Ajuste de la matriz

Si no se cumple la condición anterior, encontrar el mínimo elemento no cubierto y ajustar la matriz.
